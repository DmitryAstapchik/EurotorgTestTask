Вывести данные на график не получилось, потому что простыми средствами в паттерне MVVM это сделать нельзя (у контрола отображения графиков нельзя установить свойство привязки Series, которое должно содержать линии, построенные по получаемым данным). В паттерне MVVM это можно сделать через поведение (Behavior), но это много кода, усложняет тестовый проект.
При получении курсов валют берутся только первые 10 валют, для которых есть данные. Так сделано, чтобы не делать много запросов к апи (для каждой валюты нужно делать по 1 запросу).
